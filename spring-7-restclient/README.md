# Spring 7 - Rest Client

Este projeto serve como base para:

* Explorar o uso do `RestClient`
* Implementar chamadas REST em estilo fluente
* Trabalhar com autenticação OAuth2
* Validar comportamento com testes automatizados

## Tecnologias

* Java 25 LTS
* Spring Framework 7
* Spring Boot 4.1.0-SNAPSHOT
    * Spring Web MVC (Web)
    * OAuth2 Client (Security)
    * HTTP Client (Rest Client)
* Spring Data Commons 4.1.0-M2
* Project Lombok 1.18.42 (Developer Tools)
