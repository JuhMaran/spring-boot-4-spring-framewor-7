## 📋 Pré-requisitos

Antes de iniciar, você precisa ter instalado:

- ☕ Java 25
- 📦 Maven 3.9+
- 🧠 Conhecimento básico em:
    - Spring Boot

## ⚙️ Instalação

```bash
# Clone o repositório
git clone https://github.com/JuhMaran/spring-boot-4-spring-framewor-7.git

# Acesse a pasta
cd spring-boot-4-spring-framewor-7/spring-7-rest-mvc-api

# Compile o projeto
mvn clean install
````

A aplicação será iniciada em:

```
http://localhost:
```

<dependency>
