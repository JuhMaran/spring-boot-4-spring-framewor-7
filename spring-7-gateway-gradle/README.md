# Spring 7 - API Gateway - Gradle

Esta aplicação é um exemplo de como utilizar o Gradle.

---

## Resumo

O **Spring Boot Gradle Plugin** fornece recursos essenciais para projetos Spring Boot:

* Integração com o **processo de build do Gradle**
* Criação de **JAR executável**
* Execução da aplicação via `bootRun`
* Gerenciamento de dependências Spring
* Inclusão de **metadados de build**

Ele funciona em conjunto principalmente com dois plugins:

* `java`
* `org.springframework.boot`

Essa integração permite que o Gradle produza **artefatos executáveis completos para aplicações Spring Boot**.
